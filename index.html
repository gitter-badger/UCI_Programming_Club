UCI Programming Club

Welcome to the Fall 2015 UCI Programming Club Website!

We are a small group of programmers and soon-to-be programmers at UCI. Programming, at its core, is problem solving. Programmers take difficult problems and solve them by transforming them into a language a computer can understand. Like all problem solving, the answers are easier to find if you pool your knowledge and creativity in a group. Our programming club is designed so that we can pose problems we are facing in lab to the group, and as a group we can brain storm tools, techniques, and the overall path towards a solution. If the problem is interesting, we may even write a few lines of code for you. :) Our club is made for all members of the UCI neuroscience community; faculty, staff, postdocs, grad students, and undergrads are all welcomed and encouraged to attend.

We encourage people who have never programmed before to attend. We will help you set up a programming environment, point out online resources, and answer any questions you have about programming in general or the specifics of any problem you have to help you get up to speed programming Python.

Example topics we have covered/may cover:

  • Text Parsing
  • Opening, modifying, and saving excel spreadsheets
  • Cell Counting
  • Mouse Tracking. Here is a mouse tracker Kyle wrote. It started as a programming club problem.
  • Spike sorting
  • Modelling a Hodgkin-Huxley action potential
  • Creating artificial neural networks
  • Programming a self-evolving software using multiple python instances
  • Modelling Hippocampal CA3

Please email Conor Cox at cdcox1@gmail.com if you'd like to be on the mailing list for reminders and additional resources, or if you have topics/projects you'd like to cover.
